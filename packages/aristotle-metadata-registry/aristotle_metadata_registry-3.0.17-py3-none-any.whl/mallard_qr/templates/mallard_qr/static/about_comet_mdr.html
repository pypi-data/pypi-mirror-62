Comet Indicator Registry

The Comet Indicator Registry is an extension set for the Aristotle MetaData Registry that provides additional items necessary for tracking the performance of entities in the health context.

Comet adds the following administed items:

  • Indicators
  • Indicator Sets
  • Outcome Areas
  • Quality Statements
Comet also adds the following unmanaged items:
  • Frameworks
